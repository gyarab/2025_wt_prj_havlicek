## Running Results Database

Tento projekt představuje webovou aplikaci sloužící jako databáze atletických výsledků. Cílem aplikace je umožnit evidenci a prohlížení informací o <span style="text-decoration:underline">běžcích</span>, <span style="text-decoration:underline">závodech</span> a jejich <span style="text-decoration:underline">výsledcích</span>. Atletické závody generují velké množství dat a jejich organizace v databázovém systému umožňuje efektivní vyhledávání, analýzu a správu těchto informací.

Základními entitami systému jsou <span style="text-decoration:underline">běžec</span>, <span style="text-decoration:underline">závod</span> a <span style="text-decoration:underline">výsledek</span>. Každý závod obsahuje informace jako <span style="text-decoration:underline">název závodu</span>, <span style="text-decoration:underline">datum konání</span>, <span style="text-decoration:underline">místo konání</span> a <span style="text-decoration:underline">délku tratě</span>. Každý běžec je reprezentován atributy jako <span style="text-decoration:underline">jméno</span>, <span style="text-decoration:underline">příjmení</span>, <span style="text-decoration:underline">datum narození</span> a <span style="text-decoration:underline">klub</span>. Entita výsledek propojuje běžce a závod a obsahuje údaje o <span style="text-decoration:underline">čase výkonu</span> a <span style="text-decoration:underline">umístění</span>.

Systém rozlišuje několik rolí uživatelů. <span style="text-decoration:underline">Anonymní návštěvník</span> může prohlížet seznam závodů a výsledků. <span style="text-decoration:underline">Registrovaný uživatel</span> může přidávat nové výsledky nebo komentáře k závodům. <span style="text-decoration:underline">Administrátor</span> má plná oprávnění ke správě databáze a může vytvářet, upravovat nebo mazat záznamy o běžcích, závodech a výsledcích.

Aplikace umožňuje filtrovat závody podle data nebo distance a zobrazovat výsledky jednotlivých běžců. V budoucnu může být systém rozšířen o statistiky výkonů, <span style="text-decoration:underline">osobní rekordy</span> nebo přehled <span style="text-decoration:underline">sezóny</span>.

Projekt je implementován pomocí frameworku <span style="text-decoration:underline">Django</span> a relační <span style="text-decoration:underline">databáze</span>, která využívá vazby mezi entitami typu jeden k mnoha, například mezi běžcem a jeho výsledky nebo mezi závodem a jeho účastníky.
